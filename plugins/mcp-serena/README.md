# mcp-serena

Serena MCPサーバーを提供するプラグインです。セマンティックコードインテリジェンス機能（シンボル検索、リファレンス検索、安全なリファクタリング）を利用できます。

## インストール

```bash
/plugin install mcp-serena@ai-plugins
```

## 提供機能

- **シンボル検索**: クラス、関数、メソッドの定義を検索
- **リファレンス検索**: シンボルの参照箇所を特定
- **シンボル概要**: ファイル内のシンボル一覧を取得（ファイル全体を読まずに構造を把握）
- **セマンティック編集**: シンボル単位でのコード置換・挿入
- **パターン検索**: 正規表現によるコード検索

## 環境変数

| 変数 | 説明 | デフォルト |
|------|------|-----------|
| `SERENA_HOME` | Serena設定ディレクトリ | `.serena` |

## 注意事項

- プロジェクトで使用する前に `activate_project` でアクティベートが必要です
- Serenaのメモリー機能は使用しません。知識は `docs/` に、手順は `skills/` に配置してください
- 詳細は `docs/serena-guide.md` を参照
